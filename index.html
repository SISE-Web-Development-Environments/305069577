Web Q&A


    Three main differents between web 1,2 and 3:


    • Web 1 - It is the “readable” phrase of the World Wide Web with flat data. In Web 1, there is only limited interaction between sites and web users. Web 1 is simply an information portal where users passively receive information without being given the opportunity to post reviews, comments, and feedback.

    • Web 2 - It is the “writable” phrase of the World Wide Web with interactive data. Unlike Web 1, Web 2 facilitates interaction between web users and sites, so it allows users to interact more freely with each other. Web 2 encourages participation, collaboration, and information sharing. Examples of Web 2 applications are Youtube, Wiki, Flickr, Facebook, and so on.

    • Web 3 - It is the “executable” phrase of Word Wide Web with dynamic applications, interactive services, and “machine-to-machine” interaction. Web 3 is a semantic web which refers to the future. In Web 3, computers can interpret information like humans and intelligently generate and distribute useful content tailored to the needs of users. The Web 3 is the web version which we are using now-days.
